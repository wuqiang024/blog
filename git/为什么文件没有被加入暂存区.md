# 为什么暂存文件没有被加入暂存区
***
除了当前项目下的.gitignore文件外，在mac下的目录一般的路径在 /Users/用户名 下面，文件名为 gitignore_global.（这是一个隐藏文件）
这里也有对相关文件忽略对规则。删掉影响对文件就可以了
